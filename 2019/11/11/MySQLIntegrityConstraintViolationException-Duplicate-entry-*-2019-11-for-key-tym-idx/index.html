MySQLIntegrityConstraintViolationException: Duplicate entry 29890-2019-11 for key tym_idx
发表于 2019-11-11| 更新于 2019-11-11|bug
阅读量:

现象

更新数据库报错，Error updating database. Cause: com.mysql.jdbc.exceptions.jdbc4.MySQLIntegrityConstraintViolationException: Duplicate entry ‘29890-2019-11’ for key ‘tym_idx’

原因

更新时违反了唯一索引tym_idx的规则，导致更新失败。

解决方式

更新时需要校验业务主键的唯一性。
